# Issue Hub

An application to search for issues in github for iOS + Android
 
## Installation Steps

1. clone repository
```
git clone git@github.com:shuvo0074/issueHub.git
```
2. For NPM users, run 
    ```
    npm install
    ```
    yarn users run 
    ```
    yarn install
    ```
3. For iOS, run 
    ```
    react-native run-ios
    ```
4. for android, run
    ```
    react-native run-android
    ```
